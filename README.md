# Decodificador Alura
## Descrição

Este é o decodificador proposto pela Alura na trilha iniciante em programação, para desenvolver as habilidades na prática do que aprendido com o curso.

## Tecnologias Usadas

- Javascript
- HTML5
- CSS
- VsCode
